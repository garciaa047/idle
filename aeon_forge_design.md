# AEON FORGE — Game Design Document
*Working title (changeable — alternates: CONTINUUM, LATTICE, DEMIURGE). An original incremental game fusing factory automation with cosmic-scale ascension. Built as an offline-first PWA.*

**One-line hook:** You are a self-improving intelligence that builds a production apparatus, collapses it, and re-instantiates it at ever-larger scales of reality — from quantum foam to the manufacture of universes — until you begin fabricating new scales themselves.

---

## 1. Genre analysis — lessons distilled

Concrete takeaways from the reference titles, then the principles I'm carrying into AEON FORGE.

- **Cookie Clicker** — Layered producers, each with geometric cost (≈×1.15 per purchase) and multiplicative upgrades; "golden cookie" burst events reward active play; heavenly-chip prestige. *Lesson:* a steady drip of "save vs spend" decisions + bounded active-reward events. *Pitfall it shows:* late game decays into pure idle with no decisions.
- **Universal Paperclips** — The loop changes *genre* three times (manufacturing → market/marketing → space). Tight early constraints (wire) make choices bite. *Lesson:* re-theme the loop at intervals; constrain resources so decisions matter; each phase feels like discovery.
- **Antimatter Dimensions** — Pure exponential nesting; prestige layers stack (Antimatter → Infinity → Eternity → Reality), each resets and rescales the prior layer *and adds a new mechanic* (dilation, time studies, glyphs, automator). *Lesson:* the gold standard for prestige layers that change the rules and automation you earn so solved early content gets ignored. Branching Time Studies = forced build choices.
- **Trimps** — Combat/exploration overlay, Helium prestige, portals, challenge "Spire" gates. *Lesson:* soft walls that force prestige; challenge-runs with modifiers grant permanent perks → replay. *Pitfall:* can drift toward spreadsheet grind.
- **NGU Idle** — A limited resource (energy/magic) allocated across many parallel systems = a constant optimization puzzle; offline-first; respec-friendly. *Lesson:* a shared limited resource across competing systems creates perpetual meaningful decisions; safe experimentation matters.
- **Realm Grinder** — Mutually-exclusive faction trees (good/evil/neutral); spells/excavations as the active layer; abdication prestige. *Lesson:* divergent specializations + the ability to re-spec into a different identity = strong replay.
- **Melvor Idle** — Many independent long-horizon skill tracks; offline-first; mastery for per-item incremental bonuses. *Lesson:* parallel goals so there's always a needle to move; milestone unlocks pace the grind.

**Principles baked in:** (1) layered producers w/ geometric cost + multiplicative upgrades; (2) re-theme the loop every prestige tier; (3) each prestige layer *automates the layer below* so solved content is automated, not re-grinded; (4) a shared limited allocation resource at the meta level; (5) branching, mutually-exclusive specialization trees; (6) active-play bursts that give a bounded edge (~1.5–3×, never mandatory); (7) hidden "Anomaly" unlocks rewarding experimentation; (8) offline progress first-class but curved so checking in matters; (9) soft walls that motivate prestige right as growth stalls; (10) anti-bloat: transparent multipliers, scientific notation, per-scale unit resets, no dead content.

## 2. Core gameplay loop

Four nested loops, each a longer cadence than the last.

- **Inner (seconds–minutes):** Reactors make **Energy** → Extractors mine **Matter** → Fabricators consume Energy+Matter into **Components** → Components auto-feed **Modules** → Modules feed **Engines** → Engines output **Structure** (run score + main shop currency). You buy producers/upgrades; you **Overclock** (tap to surge a line, on cooldown) and catch **Resonance** pickups (drifting particle you tap for a burst — the golden-cookie analog).
- **Mid (10s of min–hours):** Costs outrun income → you **Collapse** (soft prestige *within* a Scale): reset the production chain, mint **Singularity (σ)**, spend σ on permanent within-Scale multipliers and new fabricator tiers. Many Collapses per Scale.
- **Macro (hours–days):** Once a Scale's tech is exhausted, you **Ascend** (hard prestige): full reset, jump to the next **Scale of Reality**, which re-themes resources, adds a new core mechanic, and grants an **Automator** that runs the prior Scale's loop for you. Ascending mints **Aeons (Æ)**.
- **Meta (days–weeks):** Spend Æ in the **Constants Tree** (permanent reality-laws affecting all runs), pick a **Paradigm** (mutually-exclusive build specialization), hunt **Anomalies** (hidden mechanics), and at the very end perform **Recursion** (reset Æ for Eternal Constants) and run **Paradoxes** (modifier challenge runs).

## 3. Resources & currencies

**Run-level (reset on Ascend):**
- **Energy (E)** — base throttle. Generated by Reactors, consumed by Fabricators. Deliberately scarce vs consumers so *routing decisions stay live*.
- **Matter (M)** — raw feedstock, mined by Extractors.
- **Components (C)** — E+M → C. First compound resource.
- **Modules** — C → Modules (tier 2).
- **Engines** — Modules → Engines (tier 3).
- **Structure (S)** — top output; run score and the main shop currency.

**Within-Scale prestige:**
- **Singularity (σ)** — minted on Collapse from total Structure produced this cycle (sublinear). Buys permanent within-Scale multipliers, new tiers, light automation.

**Hard prestige:**
- **Aeons (Æ)** — minted on Ascend from Scale depth + banked σ. The meta-currency.

**Meta / side:**
- **Cosmic Constants** — permanent tree nodes bought with Æ.
- **Flux** — a *capped, active-only* currency that fills while you actively play (tapping, overclocking, catching Resonance) and drains while idle; spent on powerful temporary boosts. This is the lever that rewards active play *without punishing idlers* — idlers simply don't use it.
- **Echoes** — discovery currency from Anomalies; unlock weird permanent toggles.

Each currency has a distinct job; nothing floats unused.

## 4. Progression stages — the Scale ladder

Target: first Ascend in ~30–60 min; ~50–200 hr total. Each Scale resets units to small numbers (so you re-experience the satisfying early curve) and introduces exactly one new mechanic + one automation grant.

1. **Quantum Foam** *(tutorial)* — Reactors→Extractors→Fabricators→Structure→Collapse. New mechanic: **Overclock** tap. (~30–60 min to first Ascend.)
2. **Atomic Lattice** — full chain (Components/Modules/Engines) + **Resonance** pickups. New mechanic: **energy routing** (split energy across lines). Automator: auto-buys Quantum producers.
3. **Planetary Crust** — **Heat/Entropy**: fabricators emit Heat; excess Heat throttles output; build Radiators. New mechanic: thermal management. First **Paradigm** choice unlocks. Automator: **auto-Collapse** at a threshold you set.
4. **Stellar Core** — Fusion tiers + **Catalysts** (craft consumable mega-multipliers). New mechanic: catalyst crafting (a small optimization minigame). Automator: auto-routes energy.
5. **Galactic Web** — **multiple parallel Forges** (2–4 sites, shared meta, independent local economies). New mechanic: site specialization & allocation (NGU-style). First **Anomaly gate** typically found here. Automator: manages a chosen site.
6. **Universal Fabric** — **dual economy**: an antimatter/entropy resource that *grows while idle*, inverting the active/idle dynamic. New mechanic: matter-vs-antimatter builds. Automator: full prior-Scale automation.
7. **Transcendent** *(endgame tail, 100–200 hr)* — the loop folds: you manufacture **new Scales themselves** (a meta-factory producing the very tiers you used). New mechanic: recursive/self-referential production + **Paradox** challenge runs. Adds **Recursion** (a third hard reset → resets Æ for **Eternal Constants**, the final meta layer).

## 5. Prestige & meta-progression

Three nested resets of increasing consequence:
- **Collapse** (frequent) — reset E/M/C/Modules/Engines/Structure within the Scale; gain σ; keep σ-upgrades.
- **Ascend** (occasional) — reset σ + everything below; jump Scale; gain Æ; keep Constants, Paradigm, Automators, Anomalies.
- **Recursion** (rare, endgame) — reset Æ + Constants; gain Eternal Constants; permanently unlock the Transcendent recursive layer + meta-modifiers.

**Constants Tree** — permanent reality-laws bought with Æ, branching with mutually-exclusive nodes near the top so you *can't take everything* → build identity. Examples: *Inflation* (production scales with √time-played), *Quantum Tunneling* (Collapses instant & auto), *Conservation* (uncapped offline gains at 60% rate), *Entanglement* (parallel Forges share 30% output).

**Paradigms** (choose 1 per Ascend, swappable):
- *Expansion* — many cheap producers, weaker multipliers (wide build).
- *Density* — few producers, huge multiplicative upgrades (tall build).
- *Entropy* — embraces Heat/antimatter; high-risk high-output; idle-leaning.
- *Computation* — converts Structure into "Insight" that auto-buys upgrades & optimizes; automation/active-leaning.
Each genuinely changes play and combines with Constants for build diversity.

**Anomalies (discovery)** — hidden triggers, hinted but not spelled out, e.g.: hold Heat at 0 for 60s → "Absolute Zero" (superconductor mode); Collapse with zero σ-upgrades 10× → "Ascetic"; perfectly balanced energy ratio → "Resonant Lattice"; all producers at equal count → "Symmetry." Grant Echoes/toggles; reward curiosity; widen the strategy search space so the game isn't solved early.

**Paradoxes (endgame challenge runs)** — optional Ascends with modifiers (no automation / 2× cost / 2× heat / single producer only) granting permanent perks on completion → replay + skill expression.

---

## 6. Automation

Core anti-grind rule: **each prestige layer automates the layer below**, and automation always *trails the frontier by ~one Scale* — newest content is hands-on, solved content runs itself. Unlock ladder: manual → auto-buy prior-Scale producers (S2) → auto-Collapse at a set threshold (S3) → auto-route energy (S4) → auto-manage a site (S5) → auto-buy upgrades / auto-craft catalysts (Computation paradigm + late Constants) → auto-Ascend (very late). All surfaced in one **Automator panel** with toggles + numeric thresholds, unlocked row by row.

## 7. Balancing formulas & rationale

- **Producer cost:** `cost(n) = base · r^n`, with `r ≈ 1.12` (wide producers) to `1.18` (premium tiers). *Why:* geometric cost vs roughly-geometric income yields the proven "buy cheapest, re-evaluate periodically" loop and natural soft walls; differing `r` makes Expansion vs Density mathematically distinct.
- **Production:** tiered multiplicative — `output_k = count_k · per_unit_k · ∏(global multipliers)`. Higher tiers feed score super-linearly, so a run accelerates then plateaus; the plateau *is* the prestige signal.
- **Collapse gain:** `σ = floor( K · (S_total / S_ref)^p )`, `p ≈ 0.5`. *Why sublinear:* doubling Structure doesn't double σ → diminishing-but-positive returns → an optimal-stopping decision (skill), and no single giant run solves the game.
- **Ascend gain:** `Æ = floor( A · Scale^q · log10(1 + Σσ) )`. *Why:* both reaching deeper Scales and banking σ matter; the log keeps σ from exploding.
- **Offline progress:** `gain = rate · t_eff`, where `t_eff = T_cap · (1 − e^(−t/T_cap))` — a saturating curve (cap a few hours early, raised by Constants). *Why:* front-loaded gains reward checking in without making 24h-away strictly optimal; active sessions stay meaningful.
- **Active bonus:** Flux/Resonance give a ~1.5–3× *peak* that decays when idle — bounded so idlers stay fully viable.
- **Heat soft-cap:** `output_eff = output / (1 + (Heat / H_thresh)^2)`. *Why:* smooth penalty on runaway production → a throughput puzzle (raise `H_thresh` via Radiators, or lean Entropy); prevents trivial infinite scaling.
- **Anti-inflation:** every Scale renames/resets units to small numbers; scientific/letter notation past 1e6; "production breakdown" tooltips expose every multiplier (transparency).

## 8. Exploits, dominant strategies & pacing (self-critique)

1. **Offline abuse** (closing app to "bank") → saturating cap + Flux/Resonance/Catalyst/Heat mechanics that only progress while active.
2. **Collapse spam / prestige treadmill** → sublinear `p=0.5` + minimum-Structure gate + auto-Collapse (S3): early it's a real decision, later it's automated, never tedious.
3. **Dominant Paradigm** → couple paradigms to content type (Heat scales favor Entropy; parallel-site scales favor Computation/Expansion); Paradoxes can ban a paradigm; swap-on-Ascend so it's a per-Scale choice, not a trap.
4. **Non-binding energy constraint** → keep energy scarce; make top tiers energy-hungry so routing stays a live choice.
5. **Number bloat** → letter/scientific notation, per-Scale unit resets, transparent breakdown tooltips.
6. **Solved too early** → Anomalies (hidden triggers) + Paradox modifiers + the idle-inverting antimatter economy + mutually-exclusive Constants make the search space too large to brute-force, and reward experimentation.
7. **Dead content** → low-Scale producers stay relevant: their σ feeds Æ, Computation converts their output into Insight, and Symmetry/single-producer Anomalies & Paradoxes reference them.
8. **Pacing dead-zones** (waiting on Æ for the next Constant) → always-on parallel goals: Anomaly hunting, Paradox runs, Catalyst optimization, and a Melvor-style Achievements track granting small permanent boosts.
9. **Choice paralysis at minute 1** → strict gradual unlock; Scale 1 is *only* reactors→structure→Collapse; each system unlocks with a short discovery beat.
10. **Flux mis-tuned** → peak ~2×, fun-but-optional; it accelerates, never gates; the idle path reaches everything.

## 9. Revisions (folded in)

Net changes from the critique, all incorporated above: the saturating offline cap; the minimum-Structure Collapse gate + auto-Collapse; paradigm↔content coupling + Paradox bans; energy-scarcity tuning; transparency tooltips + notation + unit resets; the Anomaly/Paradox/dual-economy "search-space wideners"; and the always-on parallel-goal track (Achievements + Anomalies + Paradoxes) so there's never a dead stretch.

## 10. Implementation plan — phased build (for Claude Code)

Architectural spine for every phase: a single **data-driven game-state object** + a delta-time **tick** function + a **render** pass, with all content (producers, scales, upgrades, constants) defined as **data, not code**, so later phases add entries rather than rewiring logic.

- **Phase 0 — PWA scaffold:** static HTML/CSS/JS, service worker (full offline), manifest + install, localStorage save/export/import, delta-time game loop, offline-gain-on-resume.
- **Phase 1 — Scale 1 core loop:** Reactors/Extractors/Fabricators/Structure, geometric costs, Overclock tap, Collapse + σ. Goal: *fun within 5 minutes.*
- **Phase 2 — Full chain + active layer:** Components/Modules/Engines, Resonance pickups, Flux, breakdown tooltips, number formatting.
- **Phase 3 — Ascend + Scale system:** data-driven Scales, Atomic Lattice, first Automator (auto-buy), saturating offline cap.
- **Phase 4 — Heat + Paradigms:** Heat/Radiators (Planetary), Paradigm system + first choice, auto-Collapse.
- **Phase 5 — Catalysts + routing:** Catalyst crafting (Stellar), energy-routing UI, more Automator rows.
- **Phase 6 — Parallel Forges + meta:** multi-site allocation (Galactic), Constants Tree + Aeons.
- **Phase 7 — Dual economy + discovery:** antimatter economy (Universal), Anomalies (hidden triggers + Echoes), Achievements track.
- **Phase 8 — Endgame + polish:** Transcendent recursive layer + Recursion + Paradoxes + Eternal Constants; full balance pass; juice (particles, animations); accessibility; PWA install prompt.

Each phase is independently runnable and testable on an iPhone Home Screen before moving on.
